# Reviewing Artifacts

This folder stores final verification evidence for this plan.

## What belongs here

- Verification checklist files with checkbox items
- Review log files for failed checks and follow-up fixes

## Suggested files

- `verification-checklist-YYYY-MM-DD.md`
- `review-log-YYYY-MM-DD.md`

## Rule

A plan should only move from `inReview` to `finished` when all required checklist items pass.
