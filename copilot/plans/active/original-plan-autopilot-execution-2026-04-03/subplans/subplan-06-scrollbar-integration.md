# Subplan 06 - Global Scrollbar Integration

## Goal
Implement integrated scrollbar styles without layout shift and with requested visual behavior.

## Tasks
- Define global scrollbar style variants with low-opacity thumb and hover/active transitions.
- Remove legacy arrow/triangle appearance where browser allows.
- Apply stable gutter/integrated layout strategy to prevent page shift when scrollbar appears.

## Validation
- Cross-page transitions between scroll and non-scroll contexts.
- Desktop and mobile behavior sanity checks.
