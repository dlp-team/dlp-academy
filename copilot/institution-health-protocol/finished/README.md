# Finished LIAs

This folder contains completed Lifecycle Integrity Audit (LIA) instances that have passed Phase 9 cleanup and are fully archived.

---

## How a LIA Ends Up Here

1. All 9 phases of `step-checklist.md` are completed
2. `findings.md` final assessment is written
3. Global logs are updated (security registry, working features baseline, known issues, regression history)
4. Test data is cleaned from Firestore and Firebase Auth
5. The LIA folder is moved from `active/` to `finished/`
