# Subject Feature Module

This folder contains all components, hooks, and logic specific to the Subject page.

## Structure
- `Subject.jsx`: Entry point for the Subject page.
- `components/`: Subject-only presentational components (SubjectHeader, TopicGrid, etc.)
- `hooks/`: Subject-specific hooks (useSubjectManager, etc.)
- `modals/`: Subject-specific modals (if any)

## Notes
- Use only for Subject-specific logic. Reusable UI should go in `src/components/ui/` or `src/components/modules/`.
- Update this README if you add new major features or refactor the Subject module.
