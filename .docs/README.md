# .docs — DLP Academy Product Documentation

> **Purpose:** This directory is the authoritative source of truth for what DLP Academy **is**, what it **must do**, and how it **must work**. It is a living document — update it whenever the product vision, features, or workflows evolve.

---

## What This Is NOT

This is NOT technical documentation about code structure. For code-level documentation, see:
- `copilot/explanations/codebase/` — Code-level architecture docs
- `copilot/institution-health-protocol/architectures/` — Feature-domain architecture docs

This is **product documentation** — the "why", the "what", and the "how it must feel to users".
